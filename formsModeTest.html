For both lists below, the behavior is:

  • Keyboard only: Tab to selected item (first by default) and use ArrowUp/ArrowDown to navigate
  • Both screen readers: Forms mode is not enabled. Arrow keys move through the list without triggering the keydown handler on the listItems. If the list item contains div's or other elements, may place soft focus on each sub-element individually
  • VoiceOver: ArrowLeft/ArrowRight moves through list items.
  • NVDA: ArrowUp/ArrowDown moves through list items.
